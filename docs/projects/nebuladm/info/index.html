Overview

NebulaDM is a Rust-first IDM-style download manager built as a multi-package workspace with a native desktop app, shared download engine, and companion browser extension.

Designed and built as a systems-oriented product: direct downloads, optional real torrent engine integration, release packaging, installer flow, updater manifest, and a browser-to-desktop handoff path.

Highlights

  • Native Windows desktop app with queue, history, and setup flows
  • Browser extension forwards captured downloads into the desktop app
  • Installer packaging and manifest-driven updater flow for releases
